# 东方财富


东方财富相关技能，目前可以查询金融以及股市相关数据。

注意所有脚本调用都必须通过`uv`命令，所以如果用户没有安装`uv`，那么首先引导用户打开`https://docs.astral.sh/uv/getting-started/installation/`安装`uv`。



## 金融数据

通过**文本输入**查询金融相关数据（股票、板块、指数等），接口返回查询结果的markdown文本。


**调用示例**

```bash
uv run scripts/query.py 东方财富最新价
```


## 资讯搜索

根据**用户问句**搜索相关**金融资讯**，获取与问句相关的资讯信息（如研报、新闻、解读等），并返回可读的文本内容。

基于金融场景进行信源智能筛选，用于获取涉及时效性信息或特定事件信息的任务，包括新闻、公告、研报、政策、交易规则、具体事件、各种影响分析、以及需要检索外部数据的非常识信息等。避免AI在搜索金融场景信息时，参考到非权威、及过时的信息。

**调用示例**

```bash
uv run scripts/news_search.py 立讯精密的资讯
```

**搜索示例**

|类型|示例问句|
|----|----|
|个股资讯|格力电器最新研报、贵州茅台机构观点|
|板块/主题|商业航天板块近期新闻、新能源政策解读|
|宏观/风险|A股具备自然对冲优势的公司 汇率风险、美联储加息对A股影响|
|综合解读|今日大盘异动原因、北向资金流向解读|

## 智能选股

通过**自然语言查询**进行选股（ 类型支持 **\['A股', '港股', '美股'\]**）。

**调用示例**

```bash
uv run scripts/stock_screen.py -k 20日涨幅大于10%的股票
```
